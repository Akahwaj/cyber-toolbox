# Cyber Toolbox

Python-based cybersecurity learning toolkit for defensive security, lab practice, and automation.

## Features
- Password strength checker
- Hash generator
- Lab port scanner
- Subnet helper
- File integrity checker

## Purpose
This project is built for cybersecurity learning and authorized lab use only.

## Disclaimer
Use this toolkit only on systems you own or are explicitly authorized to test.

## Run
```bash
python cyber_toolbox.py
```

## Tools Included
### Password Strength Checker
Evaluates password strength and provides suggestions for improvement.

### Hash Generator
Generates MD5, SHA1, and SHA256 hashes for text input.

### Lab Port Scanner
Checks a set of common ports on a host you own or are authorized to test.

### Subnet Helper
Displays basic network details from CIDR notation.

### File Integrity Checker
Generates a SHA256 hash for a file so you can compare it later and verify changes.
